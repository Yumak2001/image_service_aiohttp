# image_service_aiohttp
Веб сервис по определению количества пикселей определенного цвета

## Установка

### _Makefile_

    $ make init

### _bash_

    $ python3 -m venv venv
    $ venv/bin/pip install -U -r requirements.txt

## Запуск

    $ . venv/bin/activate

    (venv) $ python main.py

Затем открыть http://0.0.0.0:8000
